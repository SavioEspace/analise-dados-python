# Introdução à Análise de Dados com Python

Projeto desenvolvido para praticar conceitos iniciais de análise de dados usando Python e Pandas.

## 📊 Conteúdo
- Leitura e exploração de dados com Pandas
- Análise descritiva
- Identificação de padrões e perguntas de negócio
- Uso de DataFrames

## 🗂 Estrutura do projeto
analise-dados-python/
├── Aula_01_Introducao_a_Analise_de_Dados_com_Python.ipynb
├── data/
└── vendas_brasil.csv


## ▶️ Como executar
1. Abra o notebook no Jupyter ou Google Colab
2. Execute as células na ordem
3. Certifique-se de que o arquivo `data/vendas_brasil.csv` está presente

## 🛠 Tecnologias
- Python
- Pandas
- NumPy
- Jupyter Notebook
